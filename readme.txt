1.首页文件存放在server文件夹下的index.html
2.sass存放在res下的css文件夹中
3.模块化主文件在server下的JavaScript文件夹里的mian.js文件里
4.项目的服务器化境时php根目录是mygulp
5.本项目在首页有，楼层功能和搜索功能页面数据全部使用Ajax渲染，所有页面的头部和页脚以及侧边栏都分别存放在单个HTML文件中使用jQuery的方法.load()引入。首页的活动栏目有倒计时效果。有登录注册功能在注册是会有正则的验证，不符合正则的规则无法注册，注册成功后会提示是否跳转到登录页面，也可以继续注册。打开登录页面时会将最后一次注册的账号和密码自动添加到文本框里，登录成功后会在页面的头部显示你登录的用户名。在所有页面的头部有搜索框搜索时会跳转到搜索页面，建议搜索手机品牌数据会比较多。可惜在后来修复一些bug时不幸把搜索功能破坏了，可以退回到上了版本查看。进入商品详情页时有关商品的所有信息都是有Ajax渲染的，在左侧的图片有放大镜效果和轮播图。点击加入购物车会将该商品的id存入cookie，默认是一件商品可以修改数值。添加购物车后刷新页面会在右侧的侧边栏购物车图标上显示数量，点击图标就会跳转到购物车页面。打开购物车页面会显示你刚才加入进来的商品，每一件商品都会计算小计（单价*数量）。在底部会有计算总价，总价是根据被选中商品小计的总和。在商品列表中可以直接操作商品的数量和删除。
6.该项目目前只有一个bug就是在首页多次刷新页面可能造成页面数据渲染失败。
